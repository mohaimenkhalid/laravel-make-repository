A very small laravel package to create **make:repository** command to laravel.

Prerequisites
-------------
 - **PHP** 7.1 or higher.
 - **Laravel** 6.x +
 
Installation
------------
```bash
composer require --dev mohaimen/laravel-make-repository
```

Usage
----------------
```bash
php artisan make:repository UserRepository
```
